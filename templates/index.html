O projektu

      

        H&M Analytics Dashboard koristi Gemini API za prevođenje pitanja na prirodnom
        jeziku u BigQuery SQL upite nad H&M fashion datasetom.

  • Flask — backend API

  • Gemini 1.5 Flash — text-to-SQL i interpretacija
    • 
        
  • Google BigQuery — izvršavanje upita
    • 
        
  • Plotly — interaktivni grafikoni
